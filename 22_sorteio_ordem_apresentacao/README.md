# 22 - Sorteio da ordem de apresentacao

## 🎯 Objetivo

Definir aleatoriamente a ordem de apresentacao de cinco alunos.

## 🧠 O que pratiquei

- Leitura de varios nomes
- Montagem de lista dinamica
- Embaralhamento com `random.shuffle()`
