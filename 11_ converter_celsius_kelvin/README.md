# 11 - Converter Celsius para Kelvin

## 🎯 Objetivo

Converter uma temperatura de Celsius para Kelvin.

## 🧠 O que pratiquei

- Formula `kelvin = celsius + 273.15`
- Leitura de `float`
- Saida formatada com f-string

## 🧾 Código

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/11_%20converter_celsius_kelvin/main.py)

## 🚀 Como executar


```bash
cd "11_ converter_celsius_kelvin"
python main.py
```
