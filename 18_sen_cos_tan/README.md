# 18 - Seno, cosseno e tangente

## 🎯 Objetivo

Calcular seno, cosseno e tangente de um angulo em graus.

## 🧠 O que pratiquei

- Funcoes trigonometricas do `math`
- Conversao de graus para radianos
- Formatacao com duas casas decimais

cd "18_sen_cos_tan"
python main.py
