# 27 - Analisador de frase

## 🎯 Objetivo

Contar e localizar a letra A em uma frase digitada.

## 🧠 O que pratiquei

- Metodos `.count()`, `.find()` e `.rfind()`
- Normalizacao com `.upper()` e `.strip()`
- Analise basica de texto

## 🧾 Código

Arquivo: [`main.py`](https://github.com/GustaFranz/exercicios_python/blob/main/27_analisador_frase/main.py)

## 🚀 Como executar


```bash
cd "27_analisador_frase"
python main.py
```
